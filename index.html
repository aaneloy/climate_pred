Objectives

We aim to predict future temperature trends in Canada using historical climate data.

Approach:

  • Analyze historical data (temperature, CO2 levels, emissions).
  • Preprocess the dataset to clean and transform it.
  • Apply machine learning models for prediction.

Dataset Description

The dataset contains Canadian climate data from 1900 to 2020, including:

  • Average yearly temperature (°C)
  • CO2 levels (ppm)
  • Greenhouse gas emissions (kilotons)

Year, Temperature (°C), CO2 (ppm), Emissions (kilotons)
1990, 8.2, 355.0, 540000
2000, 9.1, 369.5, 600000
2010, 9.7, 389.0, 620000
2020, 10.2, 412.5, 630000
